# Computer Club Simulation

Реализация тестового задания: симуляция работы компьютерного клуба.
Вместе с исходным кодом прикреплено несколько тестовых файлов - их можно запустить для проверки работы программы

## Requirements
- C++17 compatible compiler
- CMake >= 3.14

## Build
```bash
cmake -S . -B build
cmake --build build
```

## Run
```bash
build/compclub <input_file>
```
